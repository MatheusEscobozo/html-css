Principais formatações

Negrio / Destaque

Nesta frase temos o negrito sem semântica utilizando a tag <b>

Nesta frase temos o negrito com semântica utilizando a tag <strong>

Itálico / Ênfase

Nesta frase temos o itálico sem semântica utilizando a tag <i>

Nesta frase temos o itálico sem semântica utilizando a tag <em>

Texto marcado

Nesta frase temos o texto sendo marcado, utilizando a tag <mark>

E no outro parágrafo temos outro texto marcado

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo. Basta utilizar as tags <big> e <small> respectivamente.

Texto deletado

Podemos criar um texto excluído para indicar que ele deve ser lido, mas não considerado utilizando a tag <del>.

Texto inserido

Também é possível marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois utilizando a tag <ins>.

Outra opção seria utilizando a tag <u>, mas não possui valor semântico para o HTML
